# Patient Admission Analytics Dashboard
## About Project
Analisis data admisi 11.977 pasien rumah sakit menggunakan menggunakan Excel + Power Query + Power BI + AI-Assisted Insight + Canva across 3 periode: Maret 2025, Februari 2026, dan Maret 2026.

# Tools
- Microsoft Excel
- Power Query
- Power BI Desktop
- AI-Assisted Insight
- Canva

## Key Findings
- Total admisi tumbuh **+34,5% YoY**
- General Surgery growth tertinggi **+678%**
- BPJS dominasi **81%** payer mix
- 3 spesialisasi baru muncul di 2026

## Analisis yang Dilakukan
- Tren volume admisi per periode
- Breakdown per Specialization + Growth YoY
- Payer mix (BPJS, Corporate, Insurance)
- Referral type analysis
- Proyeksi volume Apr-Jun 2026
- 5 rekomendasi strategis untuk manajemen
